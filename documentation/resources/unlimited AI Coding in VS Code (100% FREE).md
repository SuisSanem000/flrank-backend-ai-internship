## 🛠️ Requirements & Installation

To get started, you will need to utilize the official **Open Code** extension inside VS Code [00:01:58].

### Step 1: Install the Extension
1. Open VS Code.
2. Go to the Extensions marketplace and search for **Open Code** [00:01:58].
3. Click **Install**. Once installed, the Open Code icon will reflect in your sidebar [00:02:06].

## 🔑 Provider Configurations

To achieve unlimited usage, you will generate and add free API keys from three primary providers. Inside the Open Code extension panel, you can connect them anytime using the `/connect` command [00:02:44].

### 1. Open Code Gen
* **Website:** [opencode.ai](https://opencode.ai)
* **Setup:** Create a free account and navigate to the dashboard to generate your API key [00:01:28, 00:01:52].
* **Connection:** In the extension terminal, type `/connect`, choose `Open Code Gen`, and paste your key [00:02:44].
* **Free Models Included:** Mimo V2.5, North Mini Code Free, Neitron 3 Ultra, and DeepSeek-V4 [00:03:13].

### 2. Open Router
* **Website:** [openrouter.ai](https://openrouter.ai)
* **Setup:** Create an account, go to your **Profile** section, click **API Keys**, and generate a new key [00:03:55].
* **Connection:** Type `/connect`, select `Open Router`, and paste your generated key [00:04:36].
* **Free Models Included:** Type `/model free` in the extension panel to access 20 to 30 additional free models without activating a credit card [00:04:51, 00:05:05].

### 3. Groq
* **Website:** [groq.com](https://groq.com)
* **Setup:** Sign up (e.g., using Google), navigate to the **API Keys** tab, and generate a free key [00:05:21, 00:05:51].
* **Connection:** Type `/connect`, select `Groq`, and enter the API key [00:06:05].

## 💻 Usage

Once your providers are connected, you can toggle or inspect models globally:

* **List Free Models:** Type `/model free` to view all available options active under your connected providers [00:06:26].
* **Prompting:** Simply type your coding questions or instructions directly into the interface [00:06:44].

### The Unlimited Workflow Strategy 🔄
If you ever happen to hit a temporary usage limitation on one provider:
1. **Switch Providers:** Instantly change your active model provider using the `/connect` or model selection prompt [00:07:42].
2. **Cycle Providers:** Alternating between **Open Code Gen**, **Open Router**, and **Groq** makes it virtually impossible to run out of free AI generation capabilities [00:07:48].
