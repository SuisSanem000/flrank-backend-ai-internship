# AI Workflow Audit Results

## 15 Recurring Tasks

| Task | Classification | Rationale |
| --- | --- | --- |
| Reading long documentation / API specs | Collaborate with AI | AI can summarize key points, but I need to read nuances myself. |
| Writing boilerplate code (e.g., standard CRUD endpoints) | Fully automate | Highly repetitive and predictable; AI handles this perfectly without much oversight. |
| Debugging complex, project-specific logic errors | Collaborate with AI | I understand the business logic better, but AI helps spot syntax or edge cases. |
| Reviewing security-critical code / authentication | Just me | Too risky to trust blindly; human verification is mandatory for security. |
| Writing unit tests for pure functions | Delegate to AI with review | AI writes good tests, but I must review to ensure it tests the right assertions. |
| Structuring high-level system architecture | Just me | Requires deep understanding of organizational goals and future scaling needs. |
| Drafting initial emails or project updates | Fully automate | Saves time on phrasing; I just provide bullet points and AI formats it. |
| Translating code between languages (e.g., Python to TS) | Delegate to AI with review | AI is great at syntax translation, but I must review framework-specific idioms. |
| Brainstorming variable or function names | Collaborate with AI | AI suggests good options, but I make the final stylistic choice. |
| Formatting Markdown files and tables | Fully automate | Tedious manual work that AI executes flawlessly and instantly. |
| Learning a completely new framework/concept | Collaborate with AI | I treat AI as a tutor, asking questions and having it explain concepts interactively. |
| Merging complex Git conflicts | Just me | Requires knowing which feature branch's logic takes precedence. |
| Writing SQL queries for complex joins | Delegate to AI with review | AI writes the syntax fast, but I must verify it against the actual database schema. |
| Generating dummy data for testing | Fully automate | AI excels at generating realistic JSON/CSV mock data quickly. |
| Setting up project environments (e.g., Dockerfiles) | Delegate to AI with review | AI creates a great starting point, but I need to tweak paths and specific versions. |
